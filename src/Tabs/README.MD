## Tabs 

与Tabs-Cell配合使用 

## 使用指南

### Step 1

在页面对应的json文件中引入组件

```json
{
	"usingComponents": {
		"mit-tabs": "/components/Tabs/index",
	}
}
```
### Step 2

在wxml文件里添加组件代码

```html
<mit-tabs>
    ...
</mit-tabs>
```

## API

|参数	    	  |说明  			   |类型            |默认值     |必须  | 可选参数 |
|:-----------: |:---------------:| :-------------:| :-------- | :--------: | :-----:|
| brandColor | 自定义品牌色 | String |  | false  | |
| scroll | 是否可滑动 | Boolean | false | false | |
