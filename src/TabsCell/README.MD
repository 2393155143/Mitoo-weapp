## Tabs-Cell 

与Tabs配合使用 

## 使用指南

### Step 1

在页面对应的json文件中引入组件

```json
{
	"usingComponents": {
		"mit-tabs-cell": "/components/Tabs-Cell/index"
	}
}
```
### Step 2

在wxml文件里添加组件代码

```html
<mit-tabs-cell>
    ...
</mit-tabs-cell>
```

## API

|参数	    	  |说明  			   |类型            |默认值     |必须  | 可选参数 |
|:-----------: |:---------------:| :-------------:| :-------- | :--------: | :-----:|
| active | 是否选中 | Boolean | false | false |
